### üñºÔ∏è Bonus: Authentic Gojo Anime Background

VS Code standard themes cannot natively add an image to your editor background. However, we've included a stunning, generated **Gojo Anime Background Wallpaper** inside this extension, which you can enable using the popular `Background` extension!

**How to activate your Gojo Background:**
1. Install this Anime Theme Collection bundle.
2. Install the **[Background (by shalldie)](https://marketplace.visualstudio.com/items?itemName=shalldie.background)** extension.
3. Open your VS Code `settings.json` and add the following code to link the included Gojo background:

```json
"background.customImages": [
    "file:///c:/Users/YourUsername/.vscode/extensions/anime-theme-collection-1.0.0/images/gojo_bg.png"
],
"background.style": {
    "content": "''",
    "pointer-events": "none",
    "position": "absolute",
    "z-index": "99999",
    "width": "100%",
    "height": "100%",
    "background-position": "center",
    "background-repeat": "no-repeat",
    "background-size": "cover",
    "opacity": 0.15
},
"background.useFront": false
```
*Note: Make sure to replace `YourUsername` with your actual PC username.*

## Installation

1. Open **Extensions** sidebar panel in VS Code. `View ‚Üí Extensions`
2. Search for `Anime Theme Collection`
3. Click **Install**
4. Click **Reload** to restart VS Code
5. Go to `File > Preferences > Color Theme` and select your favorite anime theme variant.
